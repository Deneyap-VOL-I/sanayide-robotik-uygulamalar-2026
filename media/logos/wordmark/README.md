# VOL-İ Yazılı Logo

Bu klasör, VOL-İ logosunun sadece yazıdan oluşan yatay wordmark sürümlerini içerir.

## Dosyalar

- `vol_i_wordmark_transparent.png`: Şeffaf arka planlı ana yazılı logo.
- `vol_i_wordmark_white_bg.png`: Beyaz arka planlı yazılı logo.
- `vol_i_wordmark_dark_bg.png`: Koyu arka planlı yazılı logo.
- `vol_i_wordmark_square_white_bg.png`: Beyaz arka planlı kare yazılı logo.
- `vol_i_wordmark_square_dark_bg.png`: Koyu arka planlı kare yazılı logo.
- `vol_i_wordmark_white_bg_source.pdn`: Beyaz arka planlı düzenlenebilir Paint.NET kaynak dosyası.
- `vol_i_wordmark_dark_bg_source.pdn`: Koyu arka planlı düzenlenebilir Paint.NET kaynak dosyası.

## Kullanım Notları

- Rapor ve açık zeminli belgelerde `vol_i_wordmark_white_bg.png` kullanılmalı.
- Koyu zeminli sunumlarda `vol_i_wordmark_dark_bg.png` kullanılmalı.
- Tasarım değişikliği gerekiyorsa ilgili `.pdn` kaynak dosyası üzerinden çalışılmalı.
