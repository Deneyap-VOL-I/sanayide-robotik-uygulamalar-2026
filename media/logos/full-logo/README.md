# VOL-İ Tam Logo

Bu klasör, VOL-İ logosunun tam logo sürümünü içerir.

## Dosyalar

- `vol_i_logo_source.png`: Orijinal kaynak görsel. Düzenleme gerektiğinde bu dosya temel alınmalı.
- `vol_i_logo_padded_transparent.png`: Şeffaf arka planlı, dikey kullanım için boşluklu sürüm.
- `vol_i_logo_square_transparent.png`: Şeffaf arka planlı kare sürüm.
- `vol_i_logo_square_white_bg.png`: Beyaz arka planlı kare sürüm.
- `vol_i_logo_square_dark_bg.png`: Koyu arka planlı kare sürüm.

## Kullanım Notları

- Rapor ve resmi belge içinde genellikle `vol_i_logo_square_white_bg.png` kullanılmalı.
- Koyu arka planlı sunumlarda `vol_i_logo_square_dark_bg.png` kullanılmalı.
- Tasarım veya yeniden çıktı alma gerektiğinde `vol_i_logo_source.png` korunmalı ve doğrudan değiştirilmemeli.
